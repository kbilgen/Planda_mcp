Planda Eval Review

İlk olarak yukarıdan bir eval rapor JSON'unu seç. Ek olarak dataset.jsonl'i de yüklersen beklenen davranışlar görünür.

Henüz veri yok

  1. Önce eval'i koştur: npm run eval:judge
  2. Rapor şuraya düşer: evals/reports/<timestamp>.json
  3. Yukarıdan o JSON'u seç
  4. Şüpheli senaryoları okuyup "Mükemmel / İyi / Orta / Kötü" seç
  5. Bitince "Dışa Aktar" ile kararlarını indir
